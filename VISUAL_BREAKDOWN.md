## ğŸ”„ Hover/Interaction States

### Buttons
```
Normal:     Dark Green (#2E7D32), shadow
Hover:      Darker shade, larger shadow, lift up
Focus:      Green border, soft glow
Active:     Pressed state with inset shadow
```

### Cards
```
Normal:     Shadow, position 0
Hover:      Larger shadow, translateY(-8px), opacity increase
```

### Links
```
Normal:     Dark Green (#2E7D32)
Hover:      Darker shade, underline
Active:     Yellow (#FFC107)
```

### Form Inputs
```
Normal:     Light border, transparent bg
Focus:      Green border, soft shadow, visible outline
Error:      Red border (if added)
```
